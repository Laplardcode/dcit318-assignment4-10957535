 Windows Forms Applications

This repository contains three simple Windows Forms applications built using C#. Each application focuses on a specific functionality:

1.Login Screen: A basic login screen that allows users to enter a username and password. When the login button is clicked, a message box is displayed showing the entered username and password.

2. Address Book: An address book application where users can enter and save contact information including name, email, and phone number.

3. Drawing Application: A simple drawing application that allows users to draw on a canvas using the mouse. Users can click and drag the mouse to draw lines or shapes on the form.

 Getting Started

To run the applications locally, you need to have Microsoft Visual Studio installed. Follow these steps:

1. Clone the repository:

```
git clone https://github.com/your-username/windows-forms-applications.git
```

2. Open the solution file in Visual Studio.

3. Build the solution to restore NuGet packages and compile the applications.

4. Run the application you want to test.

 Usage

Each application has its own form with the necessary controls. Here's a brief overview of how to use each application:

1. **Login Screen**: Enter a username and password in the respective text boxes. Click the login button to display a message box with the entered username and password.

2. **Address Book**: Enter the name, email, and phone number in their respective text boxes. Click the save button to save the entered information.

3. **Drawing Application**: Click and drag the mouse to draw lines or shapes on the canvas.
